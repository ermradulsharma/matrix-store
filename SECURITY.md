## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the Matrix Store Backend, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to the project maintainer. Note that this is a placeholder email and should be updated by the repository owner.

You can expect a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
